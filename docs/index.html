Base e leitura

O que esta pagina resume

Esta visualizacao foi montada a partir do notebook `video_game_sales_grupo/exercicio.ipynb`, usando o dataset `gregorut/videogamesales` via KaggleHub. Para manter a analise consistente, foram removidas as linhas sem `Year` ou `Publisher`, o que deixa a base final com 16.291 jogos entre 1980 e 2020.

Action lidera o acumulado

Com 1.722,84 milhoes, o genero Action aparece na frente de Sports e Shooter quando olhamos a soma de vendas globais.

PS2 domina entre plataformas

O PS2 chega a 1.233,46 milhoes e abre distancia do X360, do PS3 e do Wii no ranking historico.

America do Norte puxa a escala

Com 4.327,65 milhoes, a regiao concentra o maior volume total, enquanto o Japao mostra forca relativa em Role-Playing.

Metodo

Como esta pagina foi montada

A pagina traduz o notebook para um formato de portfolio no GitHub Pages. A ideia nao e substituir a exploracao em Python, mas deixar a leitura principal acessivel sem abrir o Jupyter.

  • Base original com 16.598 linhas e 11 colunas.
  • Base tratada com 16.291 linhas apos remover ausencias em `Year` e `Publisher`.
  • Vendas expressas em milhoes de unidades, seguindo o dataset original.
  • Notebook fonte: `video_game_sales_grupo/exercicio.ipynb`.
Leitura importante

Os anos de 2017 e 2020 aparecem com pouquissimos registros no dataset. Isso significa que a queda extrema no fim da serie deve ser lida como limite da base, e nao so como comportamento do mercado.

Ponto forte da pagina: ela vira uma vitrine visual do notebook sem depender de screenshots ou export manual de graficos.
